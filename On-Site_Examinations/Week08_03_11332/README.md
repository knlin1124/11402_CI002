# UVa 11332 - Summing Digits

## 解題思路

這題要求計算一個數字的 **Digital Root（數位根）**。

做法是將數字的每一位數字相加，如果結果仍然是兩位數以上，就繼續把每一位數字相加，直到剩下一位數為止。

輸入會一直持續，直到輸入 `0` 為止。

---

## 關鍵邏輯

### 1. 每位數字相加

利用 `% 10` 取得個位數：

```cpp
sum += n % 10;
```

再利用：

```cpp
n /= 10;
```

去掉最後一位數。

---

### 2. 重複直到剩下一位

如果第一次相加後：

```text
9875

9+8+7+5 = 29
```

29 還不是一位數，所以再繼續：

```text
2+9 = 11

1+1 = 2
```

最後答案就是：

```text
2
```

因此只要：

```cpp
while (n >= 10)
```

就持續做數位相加。

---

## 範例推導

假設輸入：

```text
9999
```

第一次：

```text
9+9+9+9 = 36
```

第二次：

```text
3+6 = 9
```

最後答案：

```text
9
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 不是只加一次，要一直加到剩一位數。
2. 輸入 `0` 時直接結束，不輸出任何東西。
3. 每次新的輸入都要重新計算。

---

## 總結（Takeaway）

這題主要練習：

1. `% 10` 取得個位數。
2. `/ 10` 去掉最後一位。
3. 使用迴圈重複計算數位和。

解題流程如下：

1. 讀入整數。
2. 若輸入為 `0`，程式結束。
3. 不斷將各位數相加。
4. 直到剩下一位數。
5. 輸出答案。

**Time Complexity：** `O(log n)`

**Space Complexity：** `O(1)`
