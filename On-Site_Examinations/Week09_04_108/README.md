# UVa 108 - Maximum Sum

## 解題思路

這題要在一個 `N x N` 的矩陣中，找出總和最大的子矩形。

子矩形必須是連續的區域，也就是選定：

```text
上界 top
下界 bottom
左界 left
右界 right
```

然後計算這塊矩形中的數字總和，找出最大值。

---

## 關鍵邏輯

### 1. 固定上下邊界

我們可以先固定子矩形的上邊界 `top` 和下邊界 `bottom`。

這樣矩形問題就可以壓縮成一維陣列問題。

---

### 2. 壓縮成一維陣列

假設固定了 `top` 到 `bottom`，就把每一欄在這幾列之間的總和加起來：

```cpp
temp[col] += matrix[bottom][col];
```

這樣 `temp` 就代表目前上下範圍內，每一欄的總和。

---

### 3. 使用 Kadane’s Algorithm

接著對 `temp` 使用最大連續子陣列演算法，也就是 Kadane’s Algorithm。

這樣就可以找出在目前上下邊界下，最大的左右範圍。

---

## 範例推導

假設矩陣是：

```text
1  -2   3
4   5  -6
-7  8   9
```

固定：

```text
top = 1
bottom = 2
```

壓縮每一欄：

```text
col0 = 4 + (-7) = -3
col1 = 5 + 8 = 13
col2 = -6 + 9 = 3
```

得到一維陣列：

```text
-3 13 3
```

最大連續子陣列是：

```text
13 + 3 = 16
```

再不斷嘗試所有上下邊界，就能找到整個矩陣的最大子矩形和。

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 矩陣中可能全部都是負數。
2. 最大值初始值不能設為 `0`。
3. 要設成很小的數，例如：

```cpp
int answer = -1000000000;
```

4. 子矩形必須是連續區域。
5. 不要暴力枚舉四個邊界再重算總和，會太慢。

---

## 總結（Takeaway）

這題是二維最大子陣列的經典題。

解題流程如下：

1. 固定上邊界 `top`。
2. 逐步增加下邊界 `bottom`。
3. 將 `top ~ bottom` 的每一欄加總成一維陣列。
4. 對一維陣列使用 Kadane’s Algorithm。
5. 更新最大答案。

**Time Complexity：** `O(N³)`

**Space Complexity：** `O(N)`
