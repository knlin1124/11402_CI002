# UVa 11615 - Family Tree

## 解題思路

這題是在一棵完整二元樹中，判斷某個節點的祖先關係。

完整二元樹的編號方式通常具有規律：

```text
節點 x 的父節點 = x / 2
```

因此只要不斷把節點往上除以 2，就可以一路找到它的父節點、祖父節點，直到根節點。

---

## 關鍵邏輯

### 1. 往上找父節點

對於節點 `x`：

```cpp
x = x / 2;
```

就可以移動到它的父節點。

---

### 2. 找共同祖先

如果要判斷兩個節點的關係，可以讓比較大的節點先往上移動。

例如：

```text
a > b
```

就讓：

```cpp
a /= 2;
```

直到兩個節點相同。

當：

```text
a == b
```

時，代表找到共同祖先。

---

## 範例推導

假設節點：

```text
10 和 6
```

往上找：

```text
10 → 5 → 2 → 1
6 → 3 → 1
```

共同祖先是：

```text
1
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 父節點是 `x / 2`。
2. 要使用整數除法。
3. 如果兩個節點深度不同，要讓較深的節點先往上移。
4. 輸入格式要依題目要求處理。

---

## 總結（Takeaway）

這題的重點是利用完整二元樹的編號規律。

解題流程如下：

1. 讀入節點。
2. 不斷往父節點移動。
3. 找出祖先或共同祖先。
4. 依照題目要求輸出結果。

**Time Complexity：** `O(log n)`

**Space Complexity：** `O(1)`
