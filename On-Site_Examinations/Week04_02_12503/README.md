# UVa 12503 - Robot Instructions

## 解題思路

這題是在模擬機器人的移動位置。

機器人一開始的位置是 `0`，接著會收到多個指令，每個指令可能是：

- `LEFT`：往左走一步，位置 `-1`
- `RIGHT`：往右走一步，位置 `+1`
- `SAME AS i`：執行第 `i` 個指令一樣的動作

最後輸出機器人所在的位置。

---

## 關鍵邏輯

### 1. 記錄每一步的實際動作

因為 `SAME AS i` 會參考之前的指令，所以不能只看文字，必須把每一步真正造成的位移存起來。

可以用陣列或 vector 記錄：

```cpp
move[i] = -1; // LEFT
move[i] = 1;  // RIGHT
```

---

## 語言差異與踩坑紀錄

這題最容易出錯的是讀取輸入。

因為指令可能是：

```text
LEFT
RIGHT
SAME AS 2
```

所以可以先讀第一個字：

```cpp
cin >> cmd;
```

如果讀到 `SAME`，再把後面的 `AS` 和數字讀掉。

---

## 總結（Takeaway）

這題的重點不是複雜演算法，而是：

1. 正確模擬每個指令。
2. 把每一步的實際位移存起來。
3. 遇到 `SAME AS i` 時，直接複製第 `i` 步的位移。

本題可以用簡單模擬解決，時間複雜度是 O(n)。
