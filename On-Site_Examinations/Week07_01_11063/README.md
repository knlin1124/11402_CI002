# UVa 11063 - B2-Sequence

## 解題思路

這題要判斷一組數列是不是 **B2-Sequence**。

B2-Sequence 需要符合兩個條件：

1. 數列中的數字必須是正整數，且要嚴格遞增。
2. 任意兩個數字相加得到的和，都不能重複。

也就是說，對所有 `i <= j`，計算：

```text
a[i] + a[j]
```

每一個結果都必須是唯一的。

---

## 關鍵邏輯

### 1. 檢查數列是否嚴格遞增

如果出現：

```cpp
a[i] <= a[i - 1]
```

代表不符合 B2-Sequence。

---

### 2. 檢查所有兩兩相加是否重複

使用 `set` 儲存出現過的和。

每次計算：

```cpp
sum = a[i] + a[j];
```

如果這個 `sum` 已經出現過，代表不是 B2-Sequence。

---

## 範例推導

假設數列是：

```text
1 2 4
```

所有可能的和：

```text
1 + 1 = 2
1 + 2 = 3
1 + 4 = 5
2 + 2 = 4
2 + 4 = 6
4 + 4 = 8
```

沒有任何重複，所以是 B2-Sequence。

---

假設數列是：

```text
1 2 3
```

所有可能的和：

```text
1 + 3 = 4
2 + 2 = 4
```

出現重複，所以不是 B2-Sequence。

---

## 語言差異與踩坑紀錄

這題容易錯的地方：

1. 必須檢查數列是否為正整數。
2. 必須檢查是否嚴格遞增。
3. 相加時要包含 `i == j` 的情況。
4. 輸出格式要完全符合題目要求：

```text
Case #1: It is a B2-Sequence.
Case #2: It is not a B2-Sequence.
```

每組測資後面還要多輸出一個空白行。

---

## 總結（Takeaway）

這題是數列性質檢查題。

解題流程如下：

1. 讀入數列。
2. 檢查數字是否為正整數且嚴格遞增。
3. 枚舉所有 `i <= j` 的組合。
4. 使用 `set` 檢查每個和是否重複。
5. 根據結果輸出是否為 B2-Sequence。

**Time Complexity：** `O(n² log n)`

**Space Complexity：** `O(n²)`
