# UVa 10242 - Fourth Point !!

## 解題思路

這題會給四個點的座標，但其中有一個點會重複出現。

我們要找出平行四邊形的第四個點。

平行四邊形有一個重要性質：

```text
A + C = B + D
```

也就是對角線中點相同。

因此只要找出「重複的點」，剩下兩個不同的點加起來，再減掉重複點，就可以得到第四個點。

---

## 關鍵邏輯

假設輸入的四個點中：

```text
P1 和 P3 是同一個點
```

那麼另外兩個點就是平行四邊形已知的兩端。

第四點可以用：

```text
answer = P2 + P4 - P1
```

其中 `P1` 是重複的點。

---

## 範例推導

假設輸入點為：

```text
(0, 0) (1, 0) (0, 0) (0, 1)
```

其中 `(0, 0)` 重複。

所以第四點為：

```text
(1, 0) + (0, 1) - (0, 0)
```

得到：

```text
(1, 1)
```

---

## 語言差異與踩坑紀錄

這題要注意：

1. 輸入是浮點數，所以要用 `double`。
2. 輸出要固定到小數點後三位。
3. 要判斷哪兩個點是重複的。
4. 公式是：

```text
不同點1 + 不同點2 - 重複點
```

不是把四個點全部加起來。

---

## 總結（Takeaway）

這題的重點是平行四邊形的向量性質。

只要找到重複點，就可以直接用向量加減求出第四點：

```text
第四點 = 另外兩點相加 - 重複點
```

本題不需要複雜幾何，只要理解座標加減即可。
