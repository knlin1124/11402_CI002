# UVa 12207 - That is Your Queue

## 解題思路

這題要模擬一個醫院的排隊系統。

一開始病人的編號為：

```text
1 2 3 ... P
```

共有 `C` 個指令：

- `N`：叫目前排在最前面的病人看診，看完後重新排到隊伍最後。
- `E x`：病人 `x` 發生緊急狀況，必須立刻到隊伍最前面。

由於需要頻繁在**隊首、隊尾**新增或刪除，因此最適合使用 **deque（雙端佇列）**。

---

## 關鍵邏輯

### 1. 初始化 Queue

如果：

```text
P = 1000000
C = 10
```

其實最多只需要：

```text
min(P, C)
```

個病人。

因為最多只會執行 `C` 次操作，多放的人永遠不可能被叫到。

所以初始化：

```cpp
for (int i = 1; i <= min(P, C); i++)
```

即可。

---

### 2. N（Next）

取出隊首：

```cpp
front()
```

輸出後：

```cpp
pop_front();
push_back();
```

代表：

```text
看診完重新排隊
```

---

### 3. E x（Emergency）

若病人已經在 Queue 中：

先移除。

再：

```cpp
push_front(x);
```

讓他直接到最前面。

---

## 範例推導

假設：

```text
Queue：

1 2 3 4
```

操作：

```text
N
```

輸出：

```text
1
```

Queue：

```text
2 3 4 1
```

---

操作：

```text
E 3
```

Queue：

```text
3 2 4 1
```

---

再：

```text
N
```

輸出：

```text
3
```

Queue：

```text
2 4 1 3
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 不需要初始化到 P，只需要：

```text
min(P, C)
```

2. `E x` 前要先把 x 從 Queue 刪掉。
3. 再放到最前面。
4. 每組測資要輸出：

```text
Case X:
```

5. 當：

```text
0 0
```

代表輸入結束。

---

## 總結（Takeaway）

這題是 deque 模擬的經典題。

解題流程如下：

1. 初始化 Queue。
2. `N`：取隊首，再放到隊尾。
3. `E x`：先刪除 x，再放到隊首。
4. 重複直到所有操作完成。

**Time Complexity：**

- `N`：`O(1)`
- `E`：`O(n)`（需要搜尋刪除）

**Space Complexity：** `O(C)`
