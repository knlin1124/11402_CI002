# UVa 11461 - Square Numbers

## 解題思路

這題要求計算區間 `[A, B]` 中，有多少個完全平方數（Perfect Square）。

完全平方數是指可以表示成：

```text
1² = 1
2² = 4
3² = 9
4² = 16
...
```

例如：

```text
A = 3
B = 9
```

區間內只有：

```text
4、9
```

因此答案就是：

```text
2
```

---

## 關鍵邏輯

### 1. 找第一個完全平方數

第一個平方數可以利用：

```cpp
ceil(sqrt(A))
```

例如：

```text
A = 10

sqrt(10) ≈ 3.16

ceil → 4
```

第一個平方數就是：

```text
4² = 16
```

---

### 2. 找最後一個完全平方數

最後一個平方數利用：

```cpp
floor(sqrt(B))
```

例如：

```text
B = 30

sqrt(30) ≈ 5.47

floor → 5
```

最後一個平方數就是：

```text
5² = 25
```

---

### 3. 計算個數

答案就是：

```text
last - first + 1
```

若：

```text
last < first
```

代表區間內沒有完全平方數，答案為：

```text
0
```

---

## 範例推導

假設：

```text
A = 3
B = 24
```

第一個平方數：

```text
ceil(sqrt(3)) = 2
```

最後一個平方數：

```text
floor(sqrt(24)) = 4
```

平方數為：

```text
2² = 4
3² = 9
4² = 16
```

共有：

```text
3 個
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 不要一個一個數字去判斷是否為平方數。
2. 要使用 `sqrt()`。
3. `ceil()` 找第一個平方根。
4. `floor()` 找最後一個平方根。
5. 當 `A = 0` 且 `B = 0` 時結束。

---

## 總結（Takeaway）

這題利用數學觀察，大幅降低時間複雜度。

解題流程如下：

1. 讀入 `A`、`B`。
2. 若兩者皆為 `0`，程式結束。
3. 利用平方根找到第一個與最後一個完全平方數。
4. 計算兩者之間共有幾個平方數。

**Time Complexity：** `O(1)`

**Space Complexity：** `O(1)`
