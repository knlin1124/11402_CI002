# UVa 673 - Parentheses Balance

## 解題思路

這題要判斷一串括號是否配對正確。

字串中只會出現：

```text
(
)
[
]
```

如果所有括號都能正確配對，輸出：

```text
Yes
```

否則輸出：

```text
No
```

最適合使用 **Stack（堆疊）** 來解決。

---

## 關鍵邏輯

### 1. 遇到左括號就放進 Stack

當讀到：

```text
(
[
```

就放入 Stack：

```cpp
st.push(ch);
```

---

### 2. 遇到右括號就檢查

如果讀到：

```text
)
```

必須符合：

```text
Stack 頂端是 (
```

如果讀到：

```text
]
```

Stack 頂端必須是：

```text
[
```

若不符合，就代表配對失敗。

---

### 3. 最後 Stack 必須為空

如果最後還有剩下：

```text
(
[
```

代表沒有成功配對。

因此：

```cpp
st.empty()
```

才算成功。

---

## 範例推導

假設：

```text
([])
```

Stack 變化：

```text
(
([
(
空
```

最後 Stack 為空，所以答案：

```text
Yes
```

---

假設：

```text
([)]
```

Stack：

```text
(
([
```

遇到：

```text
)
```

Stack 頂端卻是：

```text
[
```

配對錯誤，因此：

```text
No
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 每組測資可能是空字串。
2. 空字串也是合法答案，要輸出：

```text
Yes
```

3. 讀取輸入要使用：

```cpp
getline()
```

4. 最後一定要檢查 Stack 是否為空。

---

## 總結（Takeaway）

這題是 Stack 的經典入門題。

解題流程如下：

1. 遇到左括號放入 Stack。
2. 遇到右括號檢查是否配對。
3. 若中途配對失敗，答案就是 No。
4. 最後 Stack 必須為空。
5. 空字串也是合法括號序列。

**Time Complexity：** `O(n)`

**Space Complexity：** `O(n)`
