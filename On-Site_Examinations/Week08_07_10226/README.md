# UVa 10226 - Hardwood Species

## 解題思路

這題要統計每種樹木名稱出現的比例。

輸入會有多組測資，每一組包含多行樹木名稱，直到空白行代表該組測資結束。

最後要依照樹木名稱的字典順序輸出：

```text
樹名 百分比
```

百分比要輸出到小數點後四位。

---

## 關鍵邏輯

### 1. 使用 map 統計樹木數量

因為最後要依照樹名排序，所以可以使用：

```cpp
map<string, int> treeCount;
```

`map` 會自動按照 key，也就是樹名，由小到大排序。

---

### 2. 計算總數

每讀入一棵樹，就讓總數加一：

```cpp
total++;
```

同時更新該樹的出現次數：

```cpp
treeCount[tree]++;
```

---

### 3. 計算百分比

百分比公式為：

```text
該樹數量 / 總樹木數量 × 100
```

C++ 中可以寫成：

```cpp
double percent = (double)count / total * 100.0;
```

---

## 範例推導

假設某組測資有：

```text
Red Alder
Ash
Aspen
Ash
```

統計結果：

```text
Ash 2
Aspen 1
Red Alder 1
```

總數為：

```text
4
```

百分比：

```text
Ash = 2 / 4 × 100 = 50.0000
Aspen = 1 / 4 × 100 = 25.0000
Red Alder = 1 / 4 × 100 = 25.0000
```

因為 `map` 會依照字典順序輸出，所以結果會是：

```text
Ash 50.0000
Aspen 25.0000
Red Alder 25.0000
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 樹名可能包含空白，所以一定要用 `getline()`。
2. 第一行是測資筆數，讀完後要處理換行。
3. 每組測資之間要輸出空白行。
4. 百分比要輸出小數點後四位。
5. 空白行代表該組測資結束。

---

## 總結（Takeaway）

這題主要練習：

1. `getline()` 讀整行字串。
2. `map` 統計資料並自動排序。
3. 百分比計算與格式化輸出。

解題流程如下：

1. 讀入測資筆數。
2. 每組測資使用 `getline()` 讀樹名。
3. 用 `map` 統計每種樹出現次數。
4. 計算百分比。
5. 依照樹名排序後輸出。

**Time Complexity：** `O(n log n)`

**Space Complexity：** `O(n)`
