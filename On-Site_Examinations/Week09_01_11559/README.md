# UVa 11559 - Event Planning

## 解題思路

這題要幫一群學生安排住宿。

已知：

- `N`：學生人數
- `B`：總預算
- `H`：飯店數量
- `W`：每間飯店提供的週數

每間飯店會提供：

1. 每人住宿價格。
2. 每個星期可提供的床位數。

目標是在 **預算內** 找到可以容納所有學生的最低住宿費用。

如果沒有任何方案符合條件，就輸出：

```text
stay home
```

---

## 關鍵邏輯

### 1. 檢查床位是否足夠

每個星期都有不同床位數。

只要有一個星期：

```text
beds >= N
```

代表這間飯店可以容納所有學生。

---

### 2. 計算住宿總費用

若床位足夠：

```text
cost = price × N
```

再判斷：

```text
cost <= B
```

若符合預算，就更新目前最便宜的方案。

---

### 3. 找最小花費

利用：

```cpp
answer = min(answer, cost);
```

最後輸出最小住宿費。

若沒有任何合法方案，就輸出：

```text
stay home
```

---

## 範例推導

假設：

```text
N = 3
B = 1000
```

某飯店：

```text
price = 200
beds = 2 3 1
```

只有第二週：

```text
beds = 3
```

可以入住。

住宿費：

```text
200 × 3 = 600
```

符合預算，因此答案可以更新為：

```text
600
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 只有某一週床位足夠即可。
2. 床位足夠後，還要檢查是否超過預算。
3. 每間飯店都要比較是否更便宜。
4. 若沒有任何符合條件的方案，要輸出：

```text
stay home
```

5. 題目輸入直到 EOF。

---

## 總結（Takeaway）

這題主要練習：

1. 多層迴圈。
2. 條件判斷。
3. 找最小值。

解題流程如下：

1. 讀入所有飯店資料。
2. 檢查每週床位是否足夠。
3. 若床位足夠，再檢查預算。
4. 更新最便宜方案。
5. 輸出答案或 `stay home`。

**Time Complexity：** `O(H × W)`

**Space Complexity：** `O(1)`
