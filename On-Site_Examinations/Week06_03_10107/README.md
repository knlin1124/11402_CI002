# UVa 10107 - What is the Median?

## 解題思路

這題會不斷輸入整數，每讀入一個新的數字後，都要輸出目前所有數字的中位數。

也就是說，資料是「一個一個進來」的，每次加入新數字後，都要重新計算目前的 median。

---

## 關鍵邏輯

### 1. 每次讀入後排序

因為題目資料量不大，可以把所有讀到的數字放進 `vector`，每讀入一個新數字就排序一次。

```cpp
numbers.push_back(x);
sort(numbers.begin(), numbers.end());
```

---

### 2. 判斷目前數量是奇數還是偶數

如果目前有 `n` 個數字：

- `n` 是奇數：中位數是中間那個數字。
- `n` 是偶數：中位數是中間兩個數字的平均。

---

## 範例推導

假設輸入依序是：

```text
1
2
3
```

讀入 `1`：

```text
[1]
median = 1
```

讀入 `2`：

```text
[1, 2]
median = (1 + 2) / 2 = 1
```

讀入 `3`：

```text
[1, 2, 3]
median = 2
```

---

## 語言差異與踩坑紀錄

這題容易錯的地方：

1. 每讀入一個數字就要立刻輸出一次中位數。
2. 偶數個資料時，要取中間兩個數的平均。
3. 題目輸出是整數，所以整數除法即可。
4. 輸入直到 EOF 結束。

---

## 總結（Takeaway）

這題是動態中位數的基本題。

最簡單的做法是：

1. 使用 `vector` 儲存所有輸入數字。
2. 每次加入新數字後排序。
3. 根據目前數量輸出中位數。

雖然也可以用兩個 heap 優化，但本題使用 `vector + sort` 就足夠理解且容易實作。

**Time Complexity：** `O(n² log n)`

**Space Complexity：** `O(n)`
