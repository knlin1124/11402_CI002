# UVa 12405 - Scarecrow

## 解題思路

這題要在一條農田上放置 **稻草人（Scarecrow）**。

農田由兩種符號組成：

```text
'.' → 空地，可以放稻草人
'#' → 已有障礙物，不需要保護
```

一個稻草人可以保護：

```text
自己
左邊一格
右邊一格
```

也就是：

```text
i-1  i  i+1
```

目標是：

> **用最少的稻草人覆蓋所有 '.'。**

這題使用 **Greedy（貪心法）** 即可。

---

## 關鍵邏輯

### 1. 從左到右掃描

依序檢查每一格。

若目前是：

```text
#
```

代表不需要放稻草人，直接往下一格。

---

### 2. 遇到 '.' 就立刻放

例如：

```text
...
```

當掃描到第一個：

```text
.
```

最佳策略就是：

```text
. S .
```

把稻草人放在目前位置。

因為可以一次保護：

```text
i
i+1
i+2
```

因此放完後直接跳：

```cpp
i += 3;
```

---

### 3. 一直掃描到最後

重複上述步驟即可。

---

## 範例推導

假設：

```text
...#..
```

掃描：

第一格：

```text
.
```

放：

```text
S
```

覆蓋：

```text
...
```

直接跳三格。

剩下：

```text
#..
```

再放一個：

```text
S
```

答案：

```text
2
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. '.' 才需要放稻草人。
2. '#' 不需要處理。
3. 放完一個稻草人可以直接跳三格。
4. 不需要回頭檢查。
5. 每組測資輸出：

```text
Case X: ans
```

---

## 總結（Takeaway）

這題是 Greedy 的經典題。

解題流程如下：

1. 從左到右掃描。
2. 遇到 '.' 就放一個稻草人。
3. 放完後跳三格。
4. 最後輸出放置數量。

**Time Complexity：** `O(n)`

**Space Complexity：** `O(1)`
