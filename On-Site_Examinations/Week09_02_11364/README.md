# UVa 11364 - Parking

## 解題思路

這題要找出 Michael 停車的位置，使得他走完所有商店再回到車子的總距離最短。

由於所有商店都位於同一直線上，因此最佳策略其實很簡單：

只要停在 **最左邊商店與最右邊商店之間的任意位置**，所需走的最短距離一定都是：

```text
(最右邊商店 - 最左邊商店) × 2
```

因為無論停在哪個中間位置，都必須走到最左端與最右端各一次，最後回到車子，因此答案固定是兩端距離的兩倍。:contentReference[oaicite:0]{index=0}

---

## 關鍵邏輯

### 1. 找出最小與最大位置

讀入所有商店位置時，同時更新：

```cpp
minPos
maxPos
```

---

### 2. 計算最短距離

答案直接為：

```cpp
(maxPos - minPos) * 2
```

不需要排序，也不用模擬停車位置。

---

## 範例推導

假設商店位置：

```text
24 13 89 37
```

最左：

```text
13
```

最右：

```text
89
```

距離：

```text
89 - 13 = 76
```

來回距離：

```text
76 × 2 = 152
```

因此輸出：

```text
152
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 不需要排序，只要找最大值與最小值。
2. 不需要嘗試每個停車位置。
3. 答案永遠是：

```text
(max - min) × 2
```

4. 每組測資都要重新初始化最大值與最小值。

---

## 總結（Takeaway）

這題主要考驗的是數學觀察。

解題流程如下：

1. 讀入所有商店位置。
2. 找出最左與最右的商店。
3. 計算兩者距離。
4. 乘以 2 即為最短步行距離。:contentReference[oaicite:1]{index=1}

**Time Complexity：** `O(n)`

**Space Complexity：** `O(1)`
