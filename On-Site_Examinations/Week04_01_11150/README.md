# UVa 11150 - Cola

## 解題思路

這題的目標是計算最多可以喝到幾瓶可樂。

除了原本擁有的 `N` 瓶可樂之外，每喝完一瓶就會得到一個空瓶，而 **3 個空瓶可以換 1 瓶新可樂**。此外，題目允許向朋友借一個空瓶，只要最後能夠歸還即可，因此可以喝到比一般情況更多的可樂。

---

## 關鍵邏輯

### 1. 空瓶換可樂

每 3 個空瓶可以換 1 瓶新可樂。

喝完新可樂後，又會再得到 1 個空瓶，因此每次換領實際上只會減少 **2 個空瓶**。

### 2. 借空瓶

如果最後剩下 **2 個空瓶**，可以先向朋友借 1 個空瓶：

- 2 + 1 = 3 個空瓶
- 換到 1 瓶可樂
- 喝完得到 1 個空瓶
- 再把這個空瓶還給朋友

因此只要最後有 **2 個空瓶**，就一定還能多喝一瓶。

### 3. 數學公式

因此最多能喝到的可樂數量為：

\[
N+\left\lfloor\frac{N}{2}\right\rfloor
\]

也可以寫成：

\[
\frac{3\times N}{2}
\]

在 C++ 中直接寫：

```cpp
cout << (3 * n) / 2 << endl;
```

即可完成。

---

## 範例推導（N = 8）

原本：

```
8 瓶可樂
```

喝完：

```
8 個空瓶
```

第一次換：

```
8 個空瓶 → 換 2 瓶
總共喝了 10 瓶
剩 2 個空瓶
```

第二次：

```
2 個舊空瓶 + 2 個新空瓶 = 4 個空瓶

4 個空瓶 → 換 1 瓶

總共喝了 11 瓶
剩 1 個空瓶
```

最後：

```
1 個舊空瓶 + 1 個新空瓶 = 2 個空瓶

向朋友借 1 個空瓶

2 + 1 = 3

再換 1 瓶

喝完還朋友 1 個空瓶
```

最後總共喝到：

```
12 瓶
```

驗證公式：

```
(3 × 8) / 2 = 12
```

結果完全一致。

## 語言差異與踩坑紀錄

### 學到的觀念

- 可以使用 while 迴圈模擬整個換瓶流程。
- 觀察規律後，可以直接推導出 O(1) 的數學公式。
- C++ 的整數除法會自動捨去小數，正好符合題目的規則。
- 理解「借一瓶、還一瓶」是這題最重要的關鍵。

---

## 總結（Takeaway）

遇到空瓶換飲料問題時，可以依照以下步驟思考：

1. 確認換領比例（例如 3 換 1）。
2. 判斷是否允許借空瓶。
3. 若允許借瓶，可觀察是否能推導成數學公式。
4. 若能推導公式，通常比模擬迴圈更快、更簡潔。

本題最大的收穫是學會透過數學觀察，把原本需要模擬的流程簡化成 O(1) 的解法。
