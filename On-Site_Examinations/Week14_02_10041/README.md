# UVa 10041 - Vito's Family

## 解題思路

這題要幫 Vito 找一個住址，使他到所有親戚家的距離總和最小。

題目會給多個親戚家的門牌號碼，因為所有房子都在同一條直線上，所以問題變成：

```text
在一維座標上，找一個點，使到所有點的距離總和最小
```

這個最佳位置就是 **中位數（Median）**。

---

## 關鍵邏輯

### 1. 將地址排序

先把所有親戚家的地址由小到大排序：

```cpp
sort(street.begin(), street.end());
```

---

### 2. 選擇中位數

排序後，中間位置的數字就是最佳住址：

```cpp
median = street[n / 2];
```

選擇中位數可以讓所有距離總和最小。

---

### 3. 計算距離總和

對每個地址計算與中位數的距離：

```cpp
abs(street[i] - median)
```

並累加起來。

---

## 範例推導

假設親戚地址是：

```text
2 4 6
```

排序後：

```text
2 4 6
```

中位數：

```text
4
```

距離總和：

```text
|2 - 4| + |4 - 4| + |6 - 4|
= 2 + 0 + 2
= 4
```

所以答案是：

```text
4
```

---

## 語言差異與踩坑紀錄

這題容易犯的錯誤：

1. 不是取平均數，而是取中位數。
2. 地址要先排序。
3. 偶數個地址時，取中間兩個任一個都可以得到最小距離。
4. 每筆測資都要重新計算。
5. 距離要用絕對值。

---

## 總結（Takeaway）

這題是中位數應用的經典題。

解題流程如下：

1. 讀入測資數量。
2. 讀入所有親戚地址。
3. 排序。
4. 找中位數。
5. 計算所有地址到中位數的距離總和。

**Time Complexity：** `O(n log n)`

**Space Complexity：** `O(n)`
